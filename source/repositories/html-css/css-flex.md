## [align-content和align-items及align-self的区别](https://juejin.cn/post/6850418113792933896)
- * align-items 的设置对象是行内成员;
- * align-content 的设置对象是所有行，且只有在多行弹性盒子容器中才生效

- 单行用items
- 多行用content
