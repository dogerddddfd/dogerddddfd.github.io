Hexo速查

Hexo文档

开头添加npx

启动服务器：npx hexo server

生成静态（部署前）：npx hexo generate

部署:npx hexo deploy

清空静态:npx hexo clean

发文：npx hexo new 文章名 -p /文件夹/文件名.md

git速查

添加所有:git add .

切换分支:git checkout

添加仓库:git remote add origin 《git链接》
